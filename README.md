## üìñ [EN] About the Project

### Why is it useful? (The Story)
This project was born from a real need: **to help my uncle**, who is colorblind. When using Blender, he often struggled to distinguish between objects placed close together or those with similar material tones.

**Dalton-Contrast** solves this through a **proximity system**: it automatically assigns high-contrast colors based on object positioning. This ensures that even if two elements are adjacent, they remain clearly distinguishable.

### Key Features
- **Proximity Calculation:** Colors shift dynamically based on position (X-axis).
- **Respectful (Non-destructive):** It doesn't touch your materials, textures, or nodes. It only affects the Viewport display.
- **Multilingual:** Automatically supports English (US/UK) and Spanish.
- **Full Support:** Modes for Deuteranopia, Protanopia, Tritanopia, and Achromatopsia.
